Task Manager CLI Application
A command-line task management system built with Python that demonstrates core programming concepts including classes, file handling, data structures, and algorithms.

Features
Create Tasks: Add new tasks with title, description, category, priority, and due date
View Tasks: Display all tasks or filter by category, priority, or status
Update Tasks: Modify task details and status
Delete Tasks: Remove tasks from your list
Search: Find tasks by keyword in title or description
Overdue Detection: Automatically identify overdue tasks
Statistics: View task completion rates and breakdowns
Data Persistence: Tasks saved to JSON file automatically
Backup System: Create and restore task backups
Priority Sorting: Organize tasks by priority (High → Medium → Low)
Due Date Tracking: Track tasks due today and overdue tasks

Installation
Clone or download the project files
Ensure Python 3.6+ is installed
No external libraries required (uses only Python standard library)
Usage
Running the Application
python main.py
This opens the interactive CLI menu.

Menu Options
1.  View All Tasks           - Display all tasks sorted by priority
2.  Add New Task             - Create a new task
3.  Delete Task              - Remove a task
4.  View Tasks by Category   - Filter tasks by category
5.  View Tasks by Priority   - View tasks organized by priority
6.  View Overdue Tasks       - Show tasks past their due date
7.  View Tasks Due Today     - Show tasks due today
8.  Mark Task as Complete    - Update task status to complete
9.  Mark Task as Incomplete  - Update task status to incomplete
10. Search Tasks            - Find tasks by keyword
0.  Exit                     - Close application
